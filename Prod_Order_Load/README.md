# 공정발주 데이터 통합

네트워크 경로의 엑셀 파일에서 "작업 발주" 시트를 읽어 하나의 엑셀 파일로 통합하는 유저폼 프로그램입니다.

## 요구 사항

- Python 3.10+
- openpyxl

## 설치

```bash
pip install -r requirements.txt
```

## 실행

```bash
python app.py
```

GUI에서 **저장 경로**와 **파일명**을 확인·변경한 뒤 **통합 실행**을 누르면 됩니다.  
기본 저장 위치는 스크립트가 있는 폴더, 기본 파일명은 `공정발주내역.xlsx`입니다.

## 동작 요약

- **소스 경로**: `config.py`의 `SOURCE_FOLDER_PATHS`에 정의된 4개 폴더를 재귀 탐색합니다.
- **시트 필터**: 시트 이름에 "작업 발주"가 포함되고 "누락주의"가 없으면 처리합니다.
- **접근 실패**: 특정 경로 접근이 실패하면 로그에만 남기고 나머지 경로는 계속 처리합니다.
- **수식**: 폴더명, BOM파일명, 발행리스트 열에는 행 번호 기반 수식이 자동으로 들어갑니다.
